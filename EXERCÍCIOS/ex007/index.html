PRINCIPAIS FORMATAÇÕES

NEGRITO / DESTAQUE

O termo em destaque presente nesta frase é este utilizando a semântica <Strong>.
Antigamente era usada a tag B(não semântica).

ITÁLICO / ÊNFASE

Nesta frase temos uma palavra em itálico usando a tag I (não semântica)

Nesta frase temos um termo com Ênfase usando a tag EM (semântica)

TEXTO MARCADO

Podemos criar também um texto marcado usando a tag MARK.

TEXTO GRANDE / PEQUENO

Estamos criando nesta frase um texto grande e um texto pequeno.

A tag BIG não é semântica, já a small sim.

TEXTO DELETADO

Podemos também deletar um texto desta forma usando a tag del, para indicar que ele pode ser lido mas não considerado.

TEXTO INSERIDO

Podemos inserir um texto desta forma usando a tag ins, para dar Ênfase e indicar que ele foi adicionado depois.

TEXTO SOBRESCRITO / SUBSCRITO

O texto sobrescrito pode ser feito desta forma: x5+3, usando a tag SUP.

Já o texto subscrito pode ser feito desta forma: H2O, usando a tag sub.
